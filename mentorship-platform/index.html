Web development

Web developers are responsible for how site looks. They create the site's layout and integrate graphics, applications and other content. Long story short, they create everything that user sees on his screen, back-end developers are responsible to do everything "under the hood".

Data science

Data science is the study of data. It involves developing methods of recording, storing and analyzing data to effectively extract useful information. To do this you have to know Python, SQL databases which are also connected with web development and how machine learning works.

Programming

At its most basic, computer programming is little more than a set of instructions to facilitate specific actions. Computer programmers create instructions for a computer to act upon by writing and testing code to enable applications and software programs to operate successfully.
